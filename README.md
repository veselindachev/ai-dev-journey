# ai-dev-journey 🚀

A 5-month, hands-on journey into AI development for .NET engineers —
**7 July – 30 November 2026**, one hour of learning plus 30 minutes of
building, every day.

## Goal

Learn modern AI development (LLM APIs, prompt engineering, function
calling, RAG, agents, MCP) and apply it on a production-grade stack:
**.NET 10 · Angular · PostgreSQL + pgvector · EF Core · Azure**.

## Portfolio projects

| # | Project | What it demonstrates | Status |
|---|---------|----------------------|--------|
| 1 | **SmartExtract** — unstructured text → validated structured data | LLM integration, prompting, structured output, streaming | 🔜 planned |
| 2 | **DataTalk** — natural-language Q&A over PostgreSQL | Safe function calling, guardrails | 🔜 planned |
| 3 | **DocChat** — chat with your documents (RAG) | Embeddings, pgvector, hybrid search, citations, evaluation | 🔜 planned |
| 4 | **MCP Toolbox** — C# MCP server + agent | Semantic Kernel, agents, Model Context Protocol | 🔜 planned |

## Repository structure

- `01-hello-llm/` — first API experiments
